# 2026-03-14 记忆

## NetNotes 技能开发完成

**完成内容：**
- 开发 NetNotes（互联网笔记本）技能，实现网页文章收藏、分类、标签管理和查询功能
- 支持静态和动态网页抓取（Playwright处理反爬）
- 实现智能分类（关键词+大模型）和自动生成摘要
- 标签系统：无需预定义，输入时自动创建，支持多标签
- 预览确认流程：生成预览文件，用户确认分类和标签后保存
- 高级查询工具：支持关键词、标签、分类、日期范围组合查询

**技术要点：**
- SQLite 多表设计：articles, tags, article_tags（多对多关系）
- 网页抓取分层：requests静态 + Playwright动态
- 交互式流程：预览 → 确认 → 保存 → 入库
- 文件名清理：移除非法字符、换行符、emoji

**遇到的问题：**
- Windows 控制台 UTF-8 编码问题 → 使用 encode('gbk', errors='ignore') 处理输出
- SQL 查询表别名不一致 → 统一使用 a.* 别名
- 动态网页抓取失败 → 增加 Playwright 等待和重试机制

**经验：**
- 交互式工具比全自动更适合需要用户确认的场景
- 预览模式让用户在保存前有机会确认和修改
- 数据库设计时提前考虑多对多关系需要关联表
- Windows 环境下编码问题需要特殊处理
