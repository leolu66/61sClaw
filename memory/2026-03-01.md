# 2026-03-01 记忆

## Windows 控制台编码问题

**错误**: Python 脚本中使用 Emoji 导致 Windows 控制台报错

**系统信息**：
- 操作系统：Windows 11 64 位 (x64)
- 控制台默认编码：**GBK** (代码页 936)
- 不支持：Emoji 表情、特殊 Unicode 字符

**问题表现**：
```
UnicodeEncodeError: 'gbk' codec can't encode character '\U0001f50d'
```

**解决方案**：
1. **避免在 print 中使用 Emoji** - 改用纯文本或 ASCII 符号
2. **或者设置代码页** - 脚本开头执行 `os.system('chcp 65001 > nul')`
3. **或者设置 PYTHONIOENCODING=utf-8** - 环境变量

**推荐做法**：
- 脚本中避免使用 Emoji（💡🔍✅❌等）
- 用纯文本符号代替：`[OK]` `[ERROR]` `[INFO]` `->` `*` 等
