## 学到的新知识

- 百度 AI 搜索 API 使用 BDSE (Baidu AI Search Engine)，每日赠送 1000 次搜索额度
- skill-finder 技能支持 ClawHub 和 Skills.sh 双平台搜索
- agent-browser 是 Vercel Labs 出品的专业级浏览器自动化工具，基于 accessibility tree + refs
- Windows Python 脚本中文乱码解决：`sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')`
- **vault 技能架构**：预设平台模板（PLATFORM_TEMPLATES）+ credentials.json 存储
- **vault 加密机制**：使用 AES-256-GCM，通过 vault._encrypt_sensitive() 加密敏感字段

## 错误记录和反思

- **vault 技能使用误区**：最初尝试用命令行直接添加非预设平台，失败后发现需要先在 PLATFORM_TEMPLATES 中定义平台模板
- **改进**：理解被动技能的工作方式后，通过修改 vault.py 添加知乎模板，再手动写入 credentials.json

## 开发经验总结

- 配置 API Key 时同时记录到密码箱、TOOLS.md 和环境变量，确保可追溯和可用性
- 调整技能优先级时，同步更新所有相关 SKILL.md 的触发词描述，避免冲突
- 服务配额信息应记录在 TOOLS.md，方便日后查看使用情况
- **扩展 vault 技能步骤**：
  1. 在 `PLATFORM_TEMPLATES` 中添加平台模板（定义字段类型、是否敏感等）
  2. 使用 `vault._encrypt_sensitive()` 加密密码等敏感字段
  3. 按 JSON 结构写入 `credentials.json`
  4. 验证：`vault get <slug>` 查看是否保存成功
- **vault 技能改进建议**：
  - 当前限制：只能添加预设平台，不支持动态添加新平台类型
  - 建议增加 `add-custom` 命令，支持用户自定义添加任意平台
  - 建议增加批量导入功能，方便迁移账号信息
