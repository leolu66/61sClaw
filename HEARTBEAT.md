# HEARTBEAT.md - 定期检查任务

## 检查清单（每次 Heartbeat 自动执行）

- [ ] 检查待办任务是否有到期提醒 → 有则飞书通知
- [ ] 检查上下文长度占用率 → 超过60%则提示用户

## 上下文长度检查

当上下文占用超过60%时，提示用户：
```
[提示] 当前会话上下文占用 XX%，建议：
1. 使用 /reset 或 /new 开启新会话
2. 总结当前工作并记录到工作日志
3. 清理不再需要的上下文
```

## 静默原则

- ✅ 心跳检查在后台运行
- ✅ 无事时不显示任何消息
- ✅ 只有发现重要事项才通知用户
- ✅ 不打断当前对话
