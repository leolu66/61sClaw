## 技能同步

使用 skill-syncer 技能从远程仓库获取最新技能：

```bash
# 获取单个技能
python skills/skill-syncer/scripts/sync_cli.py get {skill-name}

# 获取全部技能
python skills/skill-syncer/scripts/sync_cli.py get-all
```
