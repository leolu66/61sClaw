## 📋 右侧首页内容区（信息聚合与待办区）

### 1. 公司基础信息栏
展示公司核心财务资质信息：
- 核算体系名称
- 纳税人识别号
- 地址、电话
- 开户行及账号

### 2. 我的待办事项模块
包含三个标签页：
- **待我审批** - 需要当前用户审批的单据
- **待我审核** - 需要当前用户审核的单据
- **我的单据** - 当前用户提交的单据

> 若无可处理单据则提示"您没有需要审批的单据"

### 3. 企业公告模块

**流程提示区**：
明确告知员工费用申请和报销的操作步骤：
- 费用申请流程：国内/国际差旅（一次差旅一次申请）、房屋/车辆租赁（一次租赁一次申请）
- 费用报销流程：发票夹（上传+查验）→ 费用报销 → 票据登记与投递（仅纸质发票）→ 业务审批 → 行政收票 → 票据派发 → 财务审核（审批）→ 出纳付款

**公告列表区**：
按时间倒序展示财务相关通知：
- 报销指南更新
- 单据追踪截止时间
- 开票信息变更
- 年终关账通知

## ✨ 整体价值

这个页面是员工进入财务系统的**首屏门户**：
- 提供全业务功能的快速入口
- 聚合个人待办和企业重要财务信息
- 让员工能高效处理工作、获取通知
- 清晰了解财务操作规范
- 提升财务流程的透明度与执行效率

## 模拟操作要点

### 常用操作流程

1. **费用报销流程**：
   发票夹（上传+查验）→ 费用报销 → 票据登记与投递 → 业务审批 → 行政收票 → 票据派发 → 财务审核 → 出纳付款

2. **费用申请流程**：
   费用申请 → 填写申请单 → 提交审批

3. **借款流程**：
   借款管理 → 填写借款单 → 提交审批 → 还款管理（后续）

## 页面详细结构

### 1. 费用申请 → 我的申请单

**导航路径：** 左侧菜单 费用申请 → 我的申请单

**页面布局：**
- 顶部操作栏：打开、退单重提、流程图（默认禁用，选中行后启用）
- 搜索区：搜索框 + 高级搜索按钮
- 数据表格：申请单列表
- 分页控件：显示 1-10, 共 X 条，共 X 页，每页显示 10/20/50/100 条

**表格列：**
| 列名 | 说明 |
|-----|------|
| 表单名称 | 申请单类型（如：国内差旅申请单） |
| 单号 | 唯一标识（如：1-SQ12026002686） |
| 报销人 | 实际报销人姓名 |
| 提单人 | 提交申请的人 |
| 描述 | 申请事由说明 |
| 项目名称 | 所属项目 |
| 部门名称 | 所属部门 |
| 单据金额 | 申请金额 |
| 审核金额 | 实际审核通过金额 |
| 核算体系名称 | 公司/核算体系 |
| 提交时间 | 申请提交时间 |
| 单据状态 | 完成/审批中/待提交等 |
| 当前审批 | 当前审批人（如有） |

**典型数据示例：**
- 表单名称：国内差旅申请单
- 单号格式：1-SQ[年月][6位序号]
- 常见状态：完成、审批中、待提交
- 常见项目：集团市场拓展项目-联通、集团市场拓展项目-移动、集团市场拓展项目-电信
