# 法定假日查询 (holiday-checker)

## 需求说明（SRS）

### 触发条件
用户说什么话会触发此技能：
- "查询假日"
- "查询2026年法定假日"
- "今天是不是假日"
- "查一下春节放假"
- "哪些日子需要上班"
- "下一个法定假日"
- "下一个假日"
- "下一个假日是什么时候"
- "最近有什么假期"

### 功能描述
查询2026年中国法定假日安排，包括：
1. 判断某天是否为假日
2. 列出某月的所有假日
3. 判断某天是否因调休而上班

### 输入/输出
- **输入**: 日期（支持多种格式如 2026-01-01、1月1日）
- **输出**: 假日信息或调休上班提示

### 依赖条件
- Python 3.x
- data/holidays.json 数据文件

### 边界情况
- 日期格式支持：YYYY-MM-DD、YYYY/MM/DD、Y月M日、YYYY年M月D日
- 默认查询年份：2026年

---

## 使用方法

### 基本用法

```bash
# 进入脚本目录
cd skills/holiday-checker/scripts

# 1. 查询某天是否为假日
python check_holiday.py 2026-01-01
python check_holiday.py 1月1日

# 2. 查询某月有哪些假日
python list_month.py 2        # 查询2月
python list_month.py 10 2026  # 查询2026年10月

# 3. 查询某天是否因调休上班
python check_workday.py 2026-01-04

# 4. 重新从网站获取数据
python fetch_holidays.py --save
```

### 支持的查询方式

| 命令 | 示例 | 说明 |
|------|------|------|
| `check_holiday.py` | `2026-01-01` | 查询指定日期是否是法定假日 |
| `check_holiday.py` | `春节` | 按假期名称查询 |
| `check_holiday.py --next` | `-` | 查询下一个法定假日 |
| `is_dayoff.py` | `今天` | 综合查询是否休息（法定假日+周末+调休） |
| `list_month.py` | `2` | 查询2月假日 |
| `check_workday.py` | `2026-01-04` | 查询是否调休上班 |

## 数据格式

```json
{
  "year": 2026,
  "source": "https://www.gov.cn/zhengce/content/202511/content_7047090.htm",
  "holidays": [
    { "name": "元旦", "date": "2026-01-01", "days": 3 },
    { "name": "春节", "date": "2026-02-15", "days": 9 }
  ],
  "workdays": [
    { "date": "2026-01-04", "note": "元旦调休" },
    { "date": "2026-02-14", "note": "春节调休" }
  ]
}
```
