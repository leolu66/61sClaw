# 五子棋游戏

打开本地五子棋HTML游戏页面，让用户可以直接在浏览器中玩五子棋。

## 游戏特性

- **棋盘大小**: 支持 11×11、15×15（标准）、19×19 三种规格
- **游戏模式**: 
  - 双人对战（PVP）- 双方获胜都放烟花
  - 人机对战（PVE）- 玩家赢放烟花，AI赢机器人跳舞
- **游戏功能**:
  - 最新落子红色标记
  - 悔棋功能
  - 重新开始
  - 平局检测
  - 胜利动画（烟花/机器人跳舞）

## 使用方法

直接调用脚本打开游戏页面：

```bash
python scripts/open_gobang.py
```

## 操作说明

1. 选择棋盘大小和游戏模式
2. 点击"开始游戏"
3. 点击棋盘交叉点落子
4. 黑棋先行，双方轮流下棋
5. 先形成五子连珠者获胜
