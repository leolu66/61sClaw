# AI新闻定时任务设置说明

## 方法一：使用PowerShell创建定时任务（推荐）

### 步骤1：打开PowerShell（管理员权限）
右键点击开始菜单 -> Windows PowerShell (管理员)

### 步骤2：运行创建脚本
```powershell
# 先进入脚本目录
cd "C:\Users\luzhe\.claude\skills\ai-news-fetcher\scripts"

# 运行设置脚本
.\setup_task.ps1
```

## 方法二：使用Windows任务计划程序手动创建

### 步骤1：打开任务计划程序
- 按 `Win + R`，输入 `taskschd.msc`，回车
- 或搜索"任务计划程序"

### 步骤2：创建基本任务
1. 点击右侧"创建基本任务"
2. 名称：`AI新闻日报获取`
3. 描述：`每天12:30自动获取AI新闻并生成日报`
4. 触发器：选择"每天"
5. 开始时间：12:30:00
6. 操作：选择"启动程序"
7. 程序/脚本：`C:\Users\luzhe\.claude\skills\ai-news-fetcher\scripts\run_ai_news.bat`
8. 完成

### 步骤3：修改任务属性（可选）
1. 找到创建的任务，右键选择"属性"
2. 在"常规"选项卡，勾选"使用最高权限运行"
3. 在"设置"选项卡，勾选：
   - 允许唤醒计算机以运行此任务
   - 如果过了计划开始时间，立即启动任务

## 方法三：使用命令行创建（schtasks）

以管理员身份运行CMD，执行：

```cmd
schtasks /create /tn "AI新闻日报获取" /tr "C:\Users\luzhe\.claude\skills\ai-news-fetcher\scripts\run_ai_news.bat" /sc daily /st 12:30 /f
```

## 如何查看和管理任务

### 查看任务
```cmd
schtasks /query /tn "AI新闻日报获取"
```

### 删除任务
```cmd
schtasks /delete /tn "AI新闻日报获取" /f
```

### 立即运行任务
```cmd
schtasks /run /tn "AI新闻日报获取"
```

## 修改定时时间

如果需要修改执行时间，可以：
1. 打开任务计划程序
2. 找到"AI新闻日报获取"任务
3. 右键 -> 属性 -> 触发器 -> 编辑
4. 修改时间为新的时间

## 常见问题

### Q: 任务运行后没有看到输出文件？
A: 检查以下几点：
1. 确认 `D:\anthropic\AI新闻\` 目录存在
2. 检查Python环境是否正常
3. 手动运行 `run_ai_news.bat` 查看错误信息

### Q: 如何查看任务运行日志？
A: 可以在任务属性中启用"将日志保存到文件"，或手动运行批处理文件查看输出。

### Q: 可以设置多个时间吗？
A: 可以，在任务属性中添加多个触发器，例如：
- 12:30 - 上午新闻
- 18:30 - 下午新闻
