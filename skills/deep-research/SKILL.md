# Deep Research 深度研究工具

基于AI和Firecrawl搜索的自动化研究系统，能够根据用户查询执行深度递归搜索，提取关键信息，生成结构化研究报告。

## 功能特性

- 🤖 自动化深度研究：多轮递归搜索，深入挖掘相关信息
- 📝 结构化输出：生成详细、结构化的研究报告
- 🔍 智能搜索：自动生成相关搜索查询，并行执行搜索
- 🎯 需求确认：通过多轮问答获取用户详细研究需求
- ⚙️ 可配置参数：支持自定义研究广度（搜索数量）和深度（递归层数）
- 💾 自动保存：报告自动保存到本地文件

## 环境要求

需要配置以下环境变量：
- `FIRECRAWL_API_KEY`：Firecrawl API 密钥（已配置）
- OpenClaw 模型调用权限

## 快速开始

### 基本使用
```bash
deep_research "你的研究主题"
```

### 自定义参数
```bash
# 指定广度（每次研究的搜索查询数量，默认7）和深度（递归层数，默认3）
deep_research "你的研究主题" --breadth 6 --depth 2
```

### 输出说明
- 研究过程中会实时展示进度
- 报告末尾自动包含所有参考来源的完整URL链接
- 默认自动保存到技能的 `output/` 目录，文件名使用报告标题自动生成（替换特殊字符为安全字符）
- 支持自定义输出路径：`--output ./my-report.md`，指定后会保存到你指定的路径

## 核心流程

1. **需求确认**：基于初始查询生成后续问题，收集用户详细需求
2. **深度研究**：
   - 自动生成相关搜索查询
   - 并行执行搜索获取网页内容
   - 提取关键信息生成学习笔记
   - 递归执行更深层次研究
3. **报告生成**：整合所有学习内容，生成结构化研究报告
4. **结果输出**：展示报告并保存到本地文件

## 配置参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `query` | 研究主题/初始查询 | 必填 |
| `--breadth` | 研究广度：每次迭代生成的搜索查询数量 | 12 |
| `--depth` | 研究深度：递归搜索的层数 | 3 |
| `--output` | 报告输出路径 | ./report.md |
| `--no-interactive` | 禁用交互模式，直接使用初始查询开始研究 | false |

## 应用场景

- 学术研究：快速收集整理领域研究资料
- 市场调研：分析市场趋势和竞争对手信息
- 技术评估：深入了解特定技术的发展和应用
- 信息收集：针对特定主题的全面信息收集整理
