# 差旅发票抓取器 (Travel Invoice Fetcher)

## 需求说明（SRS）

### 触发条件
用户说什么话会触发此技能：
- "抓取差旅发票"
- "获取出差发票"
- "下载行程发票"
- "travel invoice"
- "抓取发票"
- "获取行程发票"

### 功能描述
根据用户描述的差旅行程信息，自动：
1. 解析行程信息（出发地、目的地、日期范围）
2. 创建对应的目录结构
3. 从QQ邮箱抓取包含"发票"关键词的邮件
4. 下载邮件附件到对应目录
5. 自动解压压缩文件

### 输入/输出
- **输入**: 用户文本描述的行程信息，支持多段行程
  - 示例1: "2月8日到3月16日，南京到北京，高铁"
  - 示例2: "3月1日-3月5日上海出差，3月10日-3月15日深圳出差，都是飞机"
- **输出**: 
  - 创建的目录路径
  - 下载的发票文件列表
  - 执行日志

### 依赖条件
- Python 3.7+
- QQ邮箱IMAP访问已开启
- 邮箱凭据存储在 vault 中（key: qq_email）
- 依赖库: imaplib, email, zipfile, py7zr（可选，用于7z解压）

### 边界情况
- 邮箱连接失败: 提示检查网络和IMAP设置
- 无匹配邮件: 提示最近7天无发票邮件
- 目录已存在: 覆盖处理
- 日期解析失败: 提示支持的日期格式
- 附件下载失败: 记录错误但继续处理其他邮件

## 使用方法

### 基本用法

直接描述行程：

```
抓取差旅发票：2月8日到3月16日，南京到北京，高铁
```

或：

```
获取出差发票：3月1日-3月5日上海出差，3月10日-3月15日深圳出差，都是飞机
```

### 支持的日期格式

- "2月8日" / "3月16日"
- "2025-02-08" / "2025-03-16"
- "02-08" / "03-16"
- "2/8" / "3/16"
- "3月1日-3月5日" / "3月1日到3月5日"

### 目录结构

创建在 `D:\Users\luzhe\报销凭证` 下：

```
报销凭证/
├── 南京-北京-0208-0316/
│   ├── 发票1.pdf
│   ├── 发票2.pdf
│   └── ...
└── 上海-深圳-0301-0315/
    ├── 发票1.pdf
    └── ...
```

## 注意事项

1. **邮箱配置**: 确保QQ邮箱已开启IMAP服务，并生成授权码
2. **凭据存储**: 邮箱账号和密码/授权码需存储在 vault 中，key为 `qq_email`
3. **日期范围**: 默认搜索最近7天的邮件，如需调整请修改配置
4. **文件覆盖**: 已存在的文件会被覆盖，请提前备份重要文件
5. **网络要求**: 需要能访问 imap.qq.com

---

## 配置说明

`scripts/config.json`:

```json
{
  "base_path": "D:\\Users\\luzhe\\报销凭证",
  "imap_server": "imap.qq.com",
  "imap_port": 993,
  "search_days": 7,
  "email_subject_keywords": ["发票"],
  "auto_extract": true,
  "supported_archives": [".zip", ".rar", ".7z"]
}
```
