# 代码统计技能

统计 skills 目录下所有技能的工作成果。

## 触发方式

用户说以下指令时触发：
- "代码统计"
- "工作统计"
- "技能统计"
- "统计技能"

## 输出格式

```markdown
# 📊 Skills 工作统计报告

## 总体概况
- 技能数量: XX 个
- 总文件数: XX 个
- 总代码行数: XXX 行
- 总大小: XX MB

## 文件类型分布
- Python 文件: XX 个 (XX 行)
- JavaScript 文件: XX 个 (XX 行)
- 其他文件: XX 个

## 技能列表（按代码行数排序）
| 排名 | 技能名称 | 文件数 | 代码行数 | 大小 |
|------|---------|--------|---------|------|
| 1 | xxx | xx | xxx | xx KB |
...
```

## 统计规则

1. **技能数量**: 包含 SKILL.md 的目录数量
2. **文件数量**: 所有文件（不含 node_modules）
3. **代码行数**: 仅统计 .py, .js 文件的有效代码行数（排除空行和注释）
4. **大小统计**: 所有文件的总大小
