# Model Selector - 交互式模型选择器

提供清晰的模型清单和简单的序号选择方式，让用户快速切换模型。

## 触发条件

用户说什么话会触发此技能：
- "换模型"
- "切换模型"
- "选择模型"
- "model selector"
- "换个模型"
- "切换 AI 模型"

## 功能描述

1. **显示模型清单** - 以表格形式展示所有可用模型，带序号
2. **显示关键信息** - 价格、定位、上下文等关键信息
3. **序号选择** - 用户输入序号即可切换模型
4. **当前模型高亮** - 清晰标识当前正在使用的模型
5. **一键切换** - 切换后自动重启 Gateway 使新模型生效

## 输入/输出

- **输入**: 用户输入模型序号（如 `3`）或查看列表（直接触发）
- **输出**: 
  - 模型清单表格
  - 切换确认信息
  - 切换成功提示

## 依赖条件

- OpenClaw Gateway 服务
- `session_status` 工具用于切换模型
- 模型价格数据（来自 `model-recommender/models-capabilities.json`）

## 边界情况

- **无效序号**: 提示重新输入
- **切换失败**: 显示错误信息并提供手动切换方法
- **Gateway 未启动**: 提示先启动 Gateway

## 使用方法

### 查看模型列表

直接说"换模型"即可看到所有可用模型。

### 切换模型

看到列表后，输入序号（如 `3`）然后回车。

## 模型切换流程

1. 用户说"换模型"
2. 显示模型清单 + 当前模型
3. 用户输入序号
4. 调用 `session_status` 切换模型
5. 重启 Gateway
6. 确认切换成功
