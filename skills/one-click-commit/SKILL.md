# 一键提交技能

将三个操作整合为一键完成：写工作日志 → 写入记忆 → 同步GitHub

## 触发方式

用户说以下指令时触发：
- "提交"
- "一键提交"

## 工作流程

### 第1步：调用 work-session-logger 写工作日志

自动回顾当前会话，调用 work-session-logger 技能生成日志文件：
- 提取任务、问题、解决方案、决策等关键信息
- 敏感信息自动脱敏
- 保存到 `logs/daily/` 目录

### 第2步：总结并写入记忆

从刚生成的工作日志中提取核心内容，用简洁语言总结，写入记忆：
- **任务**：完成了什么
- **错误**：遇到什么问题
- **经验**：学到了什么

更新 `MEMORY.md`

### 第3步：同步 GitHub

执行 Git 操作：
```bash
git add -A
git commit -m "提交描述"
git push origin main
```

## 输出格式

完成后输出：

```
================================
[ 一键提交完成 ]
================================

1. 工作日志已记录
   -> logs/daily/YYYY-MM-DD-NNN-xxx.md

2. 记忆已更新
   -> MEMORY.md

3. GitHub 已同步
   -> 已推送到 main 分支
```

## 注意事项

- 如果某些步骤失败，继续执行后续步骤，最后报告错误
- Git commit 信息自动生成，格式：`feat: 日期 + 任务概述`
- 确保工作区是 git 仓库且有远程分支配置
