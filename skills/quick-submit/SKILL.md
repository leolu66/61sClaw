# Quick Submit - 快速提交技能

一键完成：写日志 → 提取记忆 → Git同步

## 触发条件

当用户说以下内容时触发：
- "提交"
- "快速提交"
- "一键提交"

## 功能说明

### 1. 写工作日志

自动调用 work-session-logger，将本次会话内容写入 `logs/daily/` 目录。

### 2. 提取重要内容到记忆

从刚写的工作日志中提取：
- 重要事件
- 经验教训
- 新规则/原则
- 关键决策

写入 `MEMORY.md`

### 3. Git 同步

检测 skills 目录是否有修改：
- 如有修改，通过 PR 推送到 GitHub main 分支

## 使用方法

直接说"提交"即可触发。
