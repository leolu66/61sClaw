## 技能开发规范

每个技能目录结构：

```
skill-name/
├── SKILL.md          # 技能说明文档（必须）
├── scripts/          # 脚本文件
├── references/       # 参考资料
├── assets/           # 静态资源
└── data/             # 运行时数据（可选）
```
