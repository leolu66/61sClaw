# 技能开发模板

**使用说明**: 复制此模板到新技能目录，根据实际情况填写。

---

## 📁 推荐目录结构

```
skills/<技能名称>/
├── SKILL.md              # 必需（使用下面的模板）
├── scripts/
│   ├── main.py           # 主脚本
│   └── config.json       # 配置文件（可选）
└── references/           # 可选（复杂技能建议有）
    ├── api_reference.md
    └── design.md
```

---

## 🎯 快速开始

1. 创建技能目录：`mkdir skills/<技能名称>`
2. 复制此模板：`cp skills/SKILL_TEMPLATE.md skills/<技能名称>/SKILL.md`
3. 填写 SKILL.md 中的 SRS 部分
4. 开发脚本代码
5. 运行测试
6. 填写 DoD 检查表
7. 提交并推送到 GitHub
